# Картинки-подсказки

Сюда кладутся изображения для заданий (мультимодальная опора, особенно для
`fill_letter` и `name_by_description`).

**Как подключить к заданию:** в JSON-айтеме добавить поле `image` с именем файла,
например `"image": "tarelka.png"`. Приложение покажет `assets/content/img/tarelka.png`
над заданием; если файла нет — просто не покажет (без ошибки).

**Источник (план):** ARASAAC — бесплатные пиктограммы для AAC/нарушений речи
(CC BY-NC-SA). Единый стиль, есть русские слова. См. BACKLOG.md.
