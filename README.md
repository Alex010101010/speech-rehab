# Тренажёр восстановления речи после инсульта

Локальное офлайн-приложение (Android-планшет) для домашних занятий по восстановлению речи.
Персона: мужчина 70+, лёгкая форма афазии; рядом жена-соведущий. Подробности — `THEORY.md`.

> ✅ **Контент прошёл валидацию логопедом** (поэлементное ревью). Сгенерирован с опорой на методики (Оппель, Клепацкая). Остаются блок-доработки (аудит пословиц на подлинность, семантическая difficulty-разметка) — см. `BACKLOG.md`.

## Структура репозитория
```
THEORY.md            — методическая проработка (6 частей)
content/             — контент-пак: json/ (для приложения) + readable/ (для логопеда) + index.json
app/                 — Flutter-приложение
  lib/               — код (модели, движок сессии, экраны, озвучка)
  assets/content/    — копия контента, вшитая в приложение
.github/workflows/   — автосборка APK + веб-превью
```

## Как получить установочный файл (.apk) — через GitHub Actions
1. Создать на GitHub **пустой** репозиторий с именем `speech-rehab` (имя важно — от него зависит адрес веб-превью).
2. Привязать и запушить:
   ```
   git remote add origin git@github.com:<логин>/speech-rehab.git
   git add . && git commit -m "MVP speech trainer"
   git push -u origin main
   ```
3. Открыть вкладку **Actions** → дождаться сборки → в результатах (Artifacts) скачать **`rech-apk`**.
4. Скопировать `app-release.apk` на планшет → разрешить «установку из неизвестных источников» → установить.

## Веб-превью (погонять без установки)
Тот же код собирается в веб-версию и публикуется на **GitHub Pages**.
Один раз включить: **Settings → Pages → Source: GitHub Actions**.
После сборки приложение доступно по адресу `https://<логин>.github.io/speech-rehab/`.
> Если репозиторий назван иначе — поправить `--base-href` в `.github/workflows/build.yml`.

## Обновление контента
Контент-пак лежит в `content/json/`. Приложение использует копию в `app/assets/content/`.
После изменения контента синхронизировать копию:
```
cp content/json/*.json content/index.json app/assets/content/
```

## Что в MVP
- Запуск сразу в дневную сессию: разминка → задания (выбор, ввод, память с озвучкой) → чтение+вопросы+пересказ → завершение на успехе.
- Крупный шрифт, большие кнопки, озвучка (русский TTS), только позитивная обратная связь.
- Кнопки для жены на пересказе («Получилось» / «Ещё раз»).
- Уровни и награды (накопительные, без наказаний), наглядный рост.
- Офлайн, читает 1362 задания из ассетов.

## Осознанно НЕ в MVP
Распознавание голоса (STT), картиночная база, кабинет логопеда. См. `THEORY.md`, ч.6.
