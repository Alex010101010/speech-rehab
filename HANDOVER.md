# Передача тренажёра на планшет — чек-лист

Версия для **контролируемого пилота**: первые сессии проходят при наблюдении
настройщика (дети/логопед) и жены. Цель чек-листа — поймать проблемы на живом
устройстве до того, как пациент останется с приложением один.

---

## 1. Собрать APK (установочник для Android)

Полная сборка APK запускается вручную (на каждый push собирается только web-превью):

- **Через сайт:** GitHub → репозиторий → вкладка **Actions** → workflow **build** →
  кнопка **Run workflow** → галка «Собрать APK» включена → **Run workflow**.
- **Через терминал:** `gh workflow run build.yml -f build_apk=true`

Сборка идёт ~5 минут. Когда зелёная — открыть этот run, внизу секция **Artifacts**,
скачать архив **`rech-apk`** (~20+ МБ). Внутри — `app-release.apk`.

> APK не подписан для Play Store — это «сторонний» установочник, ставится напрямую.

---

## 2. Доставить APK на планшет

Любой из способов (выбрать удобный):

1. **Через кабель/перенос файла:** распаковать `rech-apk`, перекинуть `app-release.apk`
   на планшет (USB, Google Drive, мессенджер «себе», почта на себя).
2. **Скачать прямо на планшете:** зайти с планшета в GitHub Actions → нужный run →
   Artifacts → `rech-apk` (нужен вход в аккаунт GitHub на планшете).

### Установка
1. Открыть `app-release.apk` через файловый менеджер на планшете.
2. Android спросит разрешение **«Установка из неизвестных источников»** для этого
   приложения (файлового менеджера/браузера) — разрешить.
3. Установить. На рабочем столе появится приложение **«Речь»**.

> Минимальная версия Android: проверить при первом запуске; если не ставится —
> прислать модель планшета и версию Android.

## 3. ОБЯЗАТЕЛЬНО до передачи пациенту (блокеры)

### 3.1 Русский голос (TTS) — критично
Без русского голоса **вся озвучка молча не работает**.

1. Открыть приложение. Если вверху главного экрана **жёлтый баннер про голос** —
   голос не установлен.
2. Установить: **Настройки планшета → Система → Язык и ввод → Синтез речи
   (Text-to-Speech) → добавить/скачать русский голос**. (Путь может чуть отличаться
   по модели — искать «Синтез речи» / «Text-to-Speech».)
3. Вернуться в приложение, перезапустить. Баннер должен исчезнуть.
4. **Проверка:** на главном нажать «Прослушать» — должна звучать русская речь.

### 3.2 Прогон одной полной сессии настройщиком
Пройти «Начать занятие» от начала до экрана «Молодец!», проверяя:
- [ ] Озвучка читает задания и слова по-русски, разборчиво, не слишком быстро.
- [ ] Кнопки крупные, нажимаются уверенно, текст не обрезается.
- [ ] **Ввод текста:** появляется экранная клавиатура, русская раскладка доступна,
      ответ можно набрать. (Если набор тяжёл — отметить: кандидат на STT в v2.)
- [ ] **Шкала длины** под полем понятна (мало / длина совпала / лишнее).
- [ ] **«Подсказка»** работает прогрессивно; **«Не знаю»** показывает ответ и пускает дальше.
- [ ] Картинки в заданиях показываются (не пустое место).
- [ ] «Отдохнуть» сохраняет прогресс и выходит; «Пропустить» переходит к следующему.

### 3.3 Картиночный режим (если у пациента тяжёлая форма)
- [ ] Главный экран → **Настройки → Картиночный режим** включается.
- [ ] Сессия идёт только узнаванием «картинка → слово» + память; картинки видны.

---

## 4. На что смотреть на первых сессиях с пациентом
- Не утомляет ли длина сессии (10 заданий) — при усталости «Отдохнуть».
- Не фрустрирует ли набор текста — если да, рассмотреть картиночный режим / ждать STT.
- Понятны ли формулировки заданий со слов жены/настройщика.
- Срабатывает ли адаптация: слишком трудные задания должны «опускаться» сами.

---

## 5. Известные ограничения текущей версии (не блокеры пилота)
- **Нет бэкапа прогресса:** сброс/потеря планшета = потеря истории занятий
  (в бэклоге как 🟡 №1 — экспорт/бэкап).
- **STT нет** (голосовой ответ) — только v2; пока ответы выбором/набором/узнаванием.
- Приложение тестировалось на web-превью; это первый прогон на реальном Android.
