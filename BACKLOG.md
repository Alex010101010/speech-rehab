# Бэклог — speech-rehab

Кандидаты из ревью по бест-практис реабилитационных приложений (Tactus Therapy, Constant Therapy, retrieval practice, WCAG, обзоры адхеренса) + фидбек методиста. Сортировка 🔴→🟡→🟢. Сделанное отслеживается в git-истории, здесь не дублируется.

| Задача | Чем критична | Приоритет | Объём |
|---|---|---|---|
| Клиническая валидация контента логопедом (вкл. открытые пункты ре-разметки: абстрактные синонимы/пословицы) | Контент — черновик; главный гейт перед реальным использованием | 🔴 | L |
| Детект доступности TTS + видимый для настраивающего фолбэк | Аудио — основной канал; на web без ru-голоса озвучка молча не работает | 🔴 | M |
| Снизить ложные «неверно» в свободном вводе (щадящее сопоставление / больше выбора вместо ввода) | Валидная форма/синоним не из `accept` → фрустрация, против errorless | 🔴 | M |
| Адаптивная лестница по навыкам (вместо единого уровня) | Афазия неравномерна по типам заданий | 🟡 | M |
