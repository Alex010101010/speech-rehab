# Бэклог — speech-rehab

Кандидаты из ревью по бест-практис реабилитационных приложений (Tactus Therapy, Constant Therapy, retrieval practice, WCAG, обзоры адхеренса) + фидбек методиста. Сортировка 🔴→🟡→🟢. Сделанное отслеживается в git-истории, здесь не дублируется.

| Задача | Чем критична | Приоритет | Объём |
|---|---|---|---|
| Детальный прогресс для логопеда/семьи + экспорт/бэкап | Сейчас только счётчики; потеря устройства = потеря прогресса | 🟡 | M |
| Расширить пул коротких текстов чтения (добавлено 6 на L1, нужно больше для разнообразия) | Floor должен быть разнообразным, не повторяться | 🟢 | M |
| Семантическая difficulty-разметка обобщений логопедом | Сложность по смыслу, не по числу слов; бот разметил 7 явно абстрактных на L3 черновиком, нужна полная проверка | 🟡 | M |
| Сократить ответы текстов чтения на L2/L3 до коротких фраз (L1 сделано) | Полные предложения-эталоны тяжелее коротких фраз | 🟢 | M |
| Визуальное ревью пака картинок ARASAAC (77 шт.) | Картинки добавлены после валидации логопедом; выборка (~16) проверена глазами, 2 заменены/убраны. Остальные — точные англ-термины, но нужен полный глазной проход перед пациентом | 🟡 | M |

| Advanced-модуль для лёгкой афазии (пословицы/паронимы/словообразование) | Вывести помеченные `advanced` типы как опциональный трек | 🟢 | M |
| STT как «запись + сравнение» (не строгое распознавание) | Голосовой ввод из THEORY; после инсульта строгое распознавание ошибается | v2 | L |
