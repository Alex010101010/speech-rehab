## Банк лёгких упражнений (для облегчения — собираем)

> Отзыв пользователя: приложение «сложновато». Цель — собрать самый доступный тир заданий (узнавание + растормаживание, уровень грубой/средней формы). Источники: методичка МОНИКИ (Щербакова/Котов, 2014), презентация Гришаниной (острый период), Клепацкая «Восстановление речи после инсульта» (PDF в открытом доступе). Каждый кандидат — либо новый L0-материал к существующему типу, либо мини-режим.

| Упражнение | Почему легче | Куда лечь | Откуда брать контент |
|---|---|---|---|
| **Договаривание словосочетаний с жёстким контекстом** — носовой→платок, сливочное→масло, проливной→дождь, минеральная→вода, шариковая→ручка | Одно высокопредсказуемое слово, контекст ведёт сам | L0 к `complete_phrase_choice` / `complete_phrase_open` или warmup | МОНИКИ (эфф.-мот., афф.-мот.), Гришанина |
| **Договаривание пословиц с жёстким концом** (опц. с показанным образцом) — «Делу время, а потехе…», «Тише едешь — дальше…» | Конец однозначен; легче, чем объяснять смысл | L0 к `proverbs` (сейчас `proverbs` = объяснение, тяжелее) | МОНИКИ (афф.-мот.), Клепацкая |
| **Автоматизированные ряды** — счёт 1→10, дни недели, месяцы; договорить ряд | Самый доступный вход, растормаживание непроизвольной речи | новый мини-режим / L0 warmup | МОНИКИ (сенсомот., эфф.-мот.), Гришанина |
| **Предмет по функции** — «на чём ездят?», «что едят?», «что носят?» | Категориальное узнавание проще описания признаков | L0 к `name_by_description` | МОНИКИ (акуст.-гност.: «определять предметы по функциям») |
| **Связать слово ↔ действие** (соотнесение) — на одной странице несколько предметов и действий, провести связи: мылом → моют, ножом → режут, ручкой → пишут, ложкой → едят, топором → рубят | Соотнесение предмет↔действие закрепляет глагольную лексику и связь предмет↔функция; узнавание из набора проще припоминания. Несколько пар на экране = меньше переходов | **Новый тип/виджет** (matching несколько пар на странице) — нужен новый APK | МОНИКИ (предмет↔функция/действие), Клепацкая |
| **Связать близкие по значению слова** (синонимы соотнесением) — на странице несколько пар, варианты уже даны, провести связи: скамья → лавка, мастер → специалист, доктор → врач, дорога → путь | Варианты предложены (узнавание), а не припоминание — легче, чем открытый `synonyms_antonyms`; соотнесение нескольких пар за раз | **Новый тип/виджет** (та же matching-механика, что и слово↔действие) — нужен новый APK | существующий пул `synonyms_antonyms` + Клепацкая |
| **Вставить пропущенную букву** (соотнесением) — на странице несколько слов с пропуском + набор букв; поставить каждую букву в своё слово: ко·ка→ш, ло·ка→д, ст·л→о | Буква выбирается из готового набора (узнавание), а не печатается; несколько слов сразу. Без клавиатуры | **Новый тип/виджет** (matching буква↔слово; расширение `fill_letter`) — нужен новый APK | существующий пул `fill_letter` (слова+картинки уже валидированы) |
| **Глобальное чтение** — подпись к картинке из 2 вариантов (не 3–4) | Меньше вариантов = ниже нагрузка | вариант `picture_word` | Гришанина (глобальное чтение), Клепацкая |

### Новые типы на будущее (НЕ для облегчения — повышают сложность, отложены)
- **Логико-грамматические обороты** (пассив/реверсив «Петю ударил Ваня — кто драчун?», причинные «…потому что…») — центральный дефект средней афазии, подтверждён 3 источниками (МОНИКИ, Гришанина, Клепацкая). Высокая ценность, но это усложнение — делать после облегчающего тира.
- **Сборка предложения из перемешанных слов** — дано предложение, слова перемешаны; собрать в правильном порядке **выбором слов касанием в нужной последовательности** (без ввода с клавиатуры). Восстановление синтаксиса/линейной схемы фразы (дефект при средней афазии). Доступность: tap-to-order, как matching-механики выше. Контент: короткие фразы (3–5 слов) из существующих текстов чтения / `complete_phrase`. Это **усложнение** (синтаксис) — после облегчающего тира; **новый виджет → новый APK**.
- Анаграммы (слова из букв); возвратные конструкции (умываю/умываюсь); омонимы, метафоры, толкование-с-ошибкой — расширения для среднего/продвинутого уровня.
